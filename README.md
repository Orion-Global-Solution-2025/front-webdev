## 📂 Estrutura do repositório

```bash
front-webdev/
├── orion-web-site/        # Projeto front-end em si
│   ├── src/
│   │   ├── assets/        # Imagens, ícones, fontes

│   ├── public/            # Arquivos estáticos públicos
│   ├── package.json       # Scripts e dependências do front
│   └── ...
```

A estrutura acima é ilustrativa e pode variar levemente de acordo com a organização interna do código.

## 🚀 Como rodar o projeto localmente

### 1. Pré-requisitos

- **Node.js** (versão LTS recomendada – 18+)
- Gerenciador de pacotes à sua escolha:
  - `npm` ou
  - `yarn` ou
  - `pnpm`

### 2. Clonar o repositório

```bash
git clone https://github.com/Orion-Global-Solution-2025/front-webdev.git
cd front-webdev/orion-web-site
```

### 3. Instalar dependências

Exemplo com **npm** (substitua por `yarn` ou `pnpm` se preferir):

```bash
npm install
```

### 4. Rodar em modo desenvolvimento

```bash
npm run dev
```
